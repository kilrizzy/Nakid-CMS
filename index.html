Nakid CMS

Nakid CMS is a stand-alone open source content management system built using PHP and CodeIgniter

Download this project as a .zip file Download this project as a tar.gz file

Nakid CMS is an open source content management system built using PHP and CodeIgniter. The goal of Nakid is to provide plenty of functionality to the user while still being very familiar and simple to a web developer.

Finally, a content management that doesn't 'steal' your website!
